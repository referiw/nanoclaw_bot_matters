# Stage1st 战报中心

基于 Quasar + Vue 3 的 Stage1st 论坛战报网站。

## 功能

- 📊 每日战报自动更新
- 📈 关键词分析和热度统计
- 💡 AI 投资建议
- 📱 响应式设计，支持移动端
- 🌓 暗色/亮色模式切换

## 技术栈

- Vue 3
- Quasar Framework
- TypeScript
- Pinia
- Netlify

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

自动部署到 Netlify：https://stage1st-report.netlify.app
